# Создание кнопки

1. Откройте [список кнопок](button_list_view.md) шаблона.
2. Нажмите кнопку «**Создать**».
3. Отобразится [конструктор кнопки](button_designer.md).
4. [Настройте][настройка-кнопки] и сохраните кнопку.
